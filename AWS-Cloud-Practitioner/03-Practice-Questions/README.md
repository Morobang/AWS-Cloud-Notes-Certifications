## Exam Format Reminder
- **65 questions** (scored) + ~15 unscored
- **90 minutes**
- **700/1000** to pass
- Multiple choice (1 correct) and multiple response (2+ correct — they tell you how many)
- No penalty for guessing — always answer every question
