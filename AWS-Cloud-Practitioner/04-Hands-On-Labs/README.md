# Hands-On Labs — AWS Cloud Practitioner

> Practical exercises to cement concepts. These use the AWS Free Tier — watch costs, but most labs here are free or near-free.

## Before You Start

- All labs assume you have an AWS account with Free Tier active.
- Use the **us-east-1 (N. Virginia)** region unless instructed otherwise — it has the widest Free Tier coverage.
- **Always clean up resources after each lab** — running EC2 instances and RDS databases cost money.
- Enable **AWS Budgets** with a $5 alert before doing any labs (Lab 1 covers this).
