# Machine Learning — Category Overview

AWS AI and Machine Learning services fall into two groups: **pre-built AI services** (call an API, get a result — no ML expertise needed) and **platform services** (build and train your own models).

For the Cloud Practitioner exam, you only need to know what each service does and when to use it — not how ML models work internally.

---

## Services in this category

| Service | What it does | Input → Output |
|---------|-------------|----------------|
| Amazon Comprehend | NLP — extract meaning from text | Text → sentiment, entities, key phrases |
| Amazon Kendra | Intelligent enterprise search | Question → relevant document answer |
| Amazon Lex | Conversational AI — chatbots and voice assistants | Speech/text → intent + response |
| Amazon Polly | Text-to-speech | Text → natural-sounding audio |
| Amazon Q | Generative AI assistant for business and developers | Question → AI-generated answer from your data |
| Amazon Rekognition | Image and video analysis | Image/video → labels, faces, text, objects |
| Amazon SageMaker AI | Build, train, and deploy ML models | Custom ML development platform |
| Amazon Textract | Extract text and structure from documents | Document image/PDF → structured text, tables, forms |
| Amazon Transcribe | Speech-to-text | Audio → text transcript |
| Amazon Translate | Language translation | Text in one language → another language |
