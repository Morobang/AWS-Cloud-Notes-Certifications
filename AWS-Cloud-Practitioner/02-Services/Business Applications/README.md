# Business Applications

AWS business application services support customer-facing communications and engagement workflows.

## Services in this category

| Service | What it does | Key use case |
|---------|-------------|--------------|
| [Amazon Connect](./Amazon%20Connect.md) | Cloud-based contact center | Customer service call centers |
| [Amazon SES](./Amazon%20Simple%20Email%20Service%20%28Amazon%20SES%29.md) | High-volume email sending | Transactional and marketing emails |

## How to choose

```
Need a full cloud contact center (phone, chat, IVR) → Amazon Connect
Need to send emails from applications at scale → Amazon SES
```
