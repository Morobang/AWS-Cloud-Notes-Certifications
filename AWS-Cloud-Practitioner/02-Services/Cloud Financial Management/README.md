# Cloud Financial Management

AWS Cloud Financial Management services help you understand, control, and optimize your AWS spending.

## Services in this category

| Service | What it does | Key use case |
|---------|-------------|--------------|
| [AWS Budgets](./AWS%20Budgets.md) | Set spend limits and receive alerts | Get notified before overspending |
| [AWS Cost Explorer](./AWS%20Cost%20Explorer.md) | Visualize and analyze historical costs | Understand where money went |
| [AWS Cost and Usage Reports](./AWS%20Cost%20and%20Usage%20Reports.md) | Detailed line-item billing export | Custom billing analytics and charge-backs |
| [AWS Marketplace](./AWS%20Marketplace.md) | Buy third-party software on AWS | Procure software billed to your AWS bill |

## How to choose

```
Want alerts before exceeding a spend limit → AWS Budgets
Need to analyze historical costs and trends → AWS Cost Explorer
Need raw line-item billing data for BI tools → Cost and Usage Reports
Want to buy or sell software on AWS → AWS Marketplace
```
